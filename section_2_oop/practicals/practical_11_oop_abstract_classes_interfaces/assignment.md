# Практическое занятие 11: ООП - абстрактные классы и интерфейсы

## Цель занятия
Научиться использовать абстрактные классы и интерфейсы в Python для создания четкой архитектуры программ и обеспечения обязательной реализации методов в дочерних классах.

## Задачи

### Задача 1: Создание абстрактного класса (25 баллов)
Создайте абстрактный класс `Shape` с:
- Абстрактным методом `area()` для вычисления площади
- Абстрактным методом `perimeter()` для вычисления периметра
- Конкретным методом `description()` для описания фигуры

### Задача 2: Реализация интерфейса (25 баллов)
Создайте интерфейс `Drawable` с методом `draw()`, который:
- Реализуется в классах фигур из задачи 1
- Выводит визуальное представление фигуры

### Задача 3: Комбинирование абстрактных классов (25 баллов)
Создайте абстрактный класс `Animal` с:
- Абстрактными методами `make_sound()` и `move()`
- Конкретным методом `sleep()`
- Реализуйте наследников: `Dog`, `Cat`, `Bird`

### Задача 4: Применение ABC в реальном примере (25 баллов)
Реализуйте систему платежей:
- Абстрактный класс `PaymentMethod` с методом `pay(amount)`
- Конкретные классы: `CreditCard`, `PayPal`, `BankTransfer`
- Обработку ошибок при неправильной сумме

## Методические указания
1. Используйте модуль `abc` для создания абстрактных классов
2. Объявляйте абстрактные методы с декоратором `@abstractmethod`
3. Не забывайте про наследование и реализацию абстрактных методов
4. Используйте аннотации типов для лучшей читаемости кода

## Требования к отчету
- Исходный код программ
- Объяснение, почему выбрана та или иная архитектура
- Примеры использования абстрактных классов
