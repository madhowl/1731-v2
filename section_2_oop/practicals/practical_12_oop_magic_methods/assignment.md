# Практическое занятие 12: ООП - магические методы

## Цель занятия
Изучить магические методы (dunder methods) в Python и научиться использовать их для придания классам специального поведения, такого как арифметические операции, сравнения, строковое представление и другие.

## Задачи

### Задача 1: Арифметические магические методы (20 баллов)
Создайте класс `Vector2D` с:
- Конструктором `__init__()` для инициализации координат x и y
- Методами `__add__()`, `__sub__()` для сложения и вычитания векторов
- Методом `__mul__()` для скалярного умножения
- Методом `__str__()` для строкового представления

### Задача 2: Методы сравнения (20 баллов)
Реализуйте класс `Student` с:
- Методами `__eq__()`, `__lt__()`, `__le__()`, `__gt__()`, `__ge__()` для сравнения студентов по среднему баллу
- Методом `__hash__()` для возможности использования в множествах и словарях

### Задача 3: Магические методы контейнеров (20 баллов)
Создайте класс `CustomList` с:
- Методами `__getitem__()`, `__setitem__()`, `__delitem__()` для доступа к элементам
- Методом `__len__()` для получения длины
- Методом `__contains__()` для проверки наличия элемента

### Задача 4: Методы для вызова и представления (20 баллов)
Реализуйте класс `Calculator` с:
- Методом `__call__()` для выполнения арифметических операций
- Методами `__repr__()` и `__str__()` для различных представлений объекта

### Задача 5: Контекстный менеджер (20 баллов)
Создайте класс `FileManager` с:
- Методами `__enter__()` и `__exit__()` для безопасной работы с файлами
- Обработкой исключений при работе с файлами

## Методические указания
1. Используйте магические методы для реализации специального поведения классов
2. Обеспечьте корректную обработку типов данных
3. Используйте аннотации типов для лучшей читаемости кода
4. Следите за тем, чтобы парные методы (например, __eq__ и __ne__) работали согласованно
