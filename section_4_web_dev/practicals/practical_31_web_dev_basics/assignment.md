### Цель занятия:
Научиться создавать простые веб-приложения с использованием HTML, CSS и базовых концепций веб-разработки.

### Задачи:
1. Создать структуру простого веб-сайта
2. Научиться использовать основные HTML-теги
3. Применить базовые стили CSS
4. Понять основы клиент-серверного взаимодействия

### План работы:
1. Создание HTML-структуры
2. Добавление стилей CSS

4. Практические задания

## 1. Создание HTML-структуры

### Пример 1: Базовая HTML-страница

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Мое первое веб-приложение</title>
</head>
<body>
    <header>
        <h1>Добро пожаловать на мой сайт</h1>
        <nav>
            <ul>
                <li><a href="#home">Главная</a></li>
                <li><a href="#about">О нас</a></li>
                <li><a href="#contact">Контакты</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <section id="home">
            <h2>Главная страница</h2>
            <p>Это пример простого веб-сайта.</p>
        </section>
        
        <section id="about">
            <h2>О нас</h2>
            <p>Информация о компании или проекте.</p>
        </section>
    </main>
    
    <footer>
        <p>&copy; 2023 Мой сайт. Все права защищены.</p>
    </footer>
</body>
</html>
```

(Остальная часть содержимого файла уже была создана ранее)
