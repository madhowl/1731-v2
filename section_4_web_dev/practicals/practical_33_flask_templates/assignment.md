# Практическое занятие 33: Flask - шаблоны

## Цель занятия
Освоить работу с HTML-шаблонами в Flask, научиться использовать систему шаблонов Jinja2 для создания динамических веб-страниц.

## Задачи
1. Изучить основы шаблонизации в Flask
2. Научиться передавать данные в шаблоны
3. Освоить использование переменных, фильтров и управляющих структур в Jinja2

## Ход работы
1. Создать Flask-приложение с использованием шаблонов
2. Создать базовый шаблон (base.html) с общей структурой страницы
3. Создать наследуемые шаблоны для разных страниц сайта
4. Реализовать передачу данных из приложения в шаблоны
5. Использовать переменные, условия и циклы в шаблонах
6. Применить фильтры для форматирования данных
7. Добавить статические файлы (CSS, JS) в проект
