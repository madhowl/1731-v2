# Курс "Разработка прикладных приложений"

## Общее описание
Этот репозиторий содержит учебные материалы для курса "Разработка прикладных приложений" для специальности 09.02.01 "Компьютерные системы и комплексы".

Курс рассчитан на 208 академических часов:
- Лекции: 70 академических часов (35 занятий по 2 часа)
- Практические занятия: 138 академических часов (69 занятий по 2 часа)


## Структура курса

Курс разделен на 6 разделов:

1. **Основы разработки приложений** - 14 академических часов (7 лекций, 8 практических)
2. **Объектно-ориентированное программирование** - 10 академических часов (3 лекции, 8 практических)
3. **Графический интерфейс пользователя (GUI)** - 30 академических часов (7 лекций, 14 практических)
4. **Веб-разработка** - 22 академических часа (5 лекций, 10 практических)
5. **Работа с базами данных** - 12 академических часов (4 лекции, 5 практических)
6. **Архитектура приложений и безопасность** - 120 академических часов (10 лекций, 34 практических)

## Структура репозитория

- `section_X_.../` - директории с материалами для каждого раздела
- `common_resources/` - общие ресурсы, руководства и справочные материалы
- `assignments/` - домашние задания и проекты

Каждая лекция и практическая работа имеют отдельную директорию соответствующими материалами, примерами кода и решениями.

## Используемые технологии

- Python (основной язык программирования)
- Tkinter и PyQt (для GUI-приложений)
- Flask и Django (для веб-разработки)
- SQLAlchemy и SQLite (для работы с базами данных)
- Git (для управления версиями)
