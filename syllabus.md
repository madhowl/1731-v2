# Учебный план курса "Разработка прикладных приложений"

## Общая информация

**Название курса:** Разработка прикладных приложений  
**Специальность:** 09.02.01 "Компьютерные системы и комплексы"  
**Общий объем:** 208 академических часов  
**Форма обучения:** очная  
**Язык программирования:** Python

## Пререквизиты

- Знание основ алгоритмизации и программирования
- Базовый уровень владения языком Python

## Цель курса

Подготовить студентов к разработке прикладных приложений различного назначения с использованием современных технологий и подходов.

## Задачи курса

- Изучить продвинутые концепции языка Python
- Освоить объектно-ориентированное программирование
- Научиться создавать графические интерфейсы пользователя
- Освоить основы веб-разработки
- Получить навыки работы с базами данных
- Изучить архитектурные паттерны и безопасность приложений
