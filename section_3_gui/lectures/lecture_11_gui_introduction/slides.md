## 1. Обзор GUI-фреймворков

Python предоставляет несколько вариантов для создания графических интерфейсов:

- **Tkinter** - встроенный в Python GUI-фреймворк
- **PyQt/PySide** - мощный фреймворк с широкими возможностями
- **Kivy** - фреймворк для кроссплатформенных приложений
- **wxPython** - фреймворк, использующий родные элементы интерфейса

### Tkinter:
- Встроен в Python
- Простой для начала
- Подходит для небольших приложений
- Ограниченные возможности кастомизации

### PyQt/PySide:
- Богатый набор виджетов
- Современный внешний вид
- Хорошая документация
- Более сложный для изучения
