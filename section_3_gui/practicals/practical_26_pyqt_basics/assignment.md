# Практическое занятие 21: Создание GUI с помощью PyQt

## Цель занятия
Изучить основы создания графических интерфейсов с использованием библиотеки PyQt, включая создание окон, виджетов, обработку событий и построение интерфейса.

## Задачи

### Задача 1: Основы PyQt (20 баллов)
Создайте приложение с основным окном:
- Используйте QApplication и QMainWindow
- Добавьте базовые виджеты (QLabel, QPushButton, QLineEdit)
- Реализуйте обработку сигналов и слотов

### Задача 2: Компоновка элементов (20 баллов)
Реализуйте различные способы компоновки:
- QVBoxLayout и QHBoxLayout для вертикальных и горизонтальных макетов
- QGridLayout для табличных макетов
- QFormLayout для макетов формы

### Задача 3: Виджеты PyQt (20 баллов)
Создайте интерфейс с различными виджетами:
- QCheckBox, QRadioButton, QComboBox
- QSpinBox, QSlider, QProgressBar
- QTableWidget, QListWidget, QTreeWidget

### Задача 4: Обработка событий (20 баллов)
Реализуйте обработку различных событий:
- Клик по кнопке, изменение текста в поле ввода
- Выбор элемента из списка или таблицы
- Изменение значения слайдера или спинбокса

### Задача 5: Комплексное приложение (20 баллов)
Разработайте приложение "Калькулятор на PyQt":
- Интерфейс с кнопками для цифр и операций
- Отображение результата
- Обработка математических операций

## Методические указания
1. Используйте классы из модуля PyQt5.QtWidgets
2. Применяйте сигналы и слоты для обработки событий
3. Используйте QLayout для компоновки элементов
4. Следите за структурой приложения и организацией кода

## Требования к отчету
- Исходный код приложений
- Скриншоты интерфейсов
- Описание архитектуры приложения
