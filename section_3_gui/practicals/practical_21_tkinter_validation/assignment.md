# Практическое занятие 21: Tkinter - валидация ввода

## Цель занятия
Научиться проверять и фильтровать пользовательский ввод в GUI-приложениях с использованием библиотеки Tkinter.

## Задачи
1. Изучить механизмы валидации ввода в Tkinter
2. Реализовать проверку данных в полях ввода
3. Обработать некорректный ввод пользователя

## Ход работы
1. Создать окно с несколькими полями ввода (например, email, телефон, возраст)
2. Реализовать валидацию для каждого поля:
   - Email: проверка формата
   - Телефон: проверка формата
   - Возраст: проверка диапазона
3. Добавить подсветку полей с ошибками
4. Реализовать кнопку отправки с общей проверкой всех полей
