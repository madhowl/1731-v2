# Практическое занятие 18: Tkinter - элементы пользовательского интерфейса

## Цель занятия
Изучить основные элементы пользовательского интерфейса в Tkinter, включая виджеты для ввода данных, отображения информации, навигации и взаимодействия с пользователем.

## Задачи

### Задача 1: Основные виджеты ввода (20 баллов)
Создайте приложение с основными виджетами ввода:
- Entry для одиночного ввода
- Text для многострочного ввода
- Spinbox для выбора из диапазона
- Combobox для выбора из списка

### Задача 2: Кнопки и переключатели (20 баллов)
Реализуйте интерфейс с:
- Кнопками разных типов (обычные, стилизованные, тумблеры)
- Checkbutton для множественного выбора
- Radiobutton для одиночного выбора
- Scale для выбора значения

### Задача 3: Отображение информации (20 баллов)
Создайте элементы для отображения информации:
- Label для текста
- Label с изображением
- Message для многострочного текста
- Canvas для графики

### Задача 4: Навигационные элементы (20 баллов)
Реализуйте навигационные элементы:
- Notebook (вкладки) для организации интерфейса
- PanedWindow для разделения области
- Menu для меню приложения
- Toolbar для инструментов

### Задача 5: Комплексное приложение (20 баллов)
Создайте приложение "Форма регистрации":
- Используйте все изученные виджеты
- Реализуйте валидацию ввода
- Добавьте систему навигации между полями

## Методические указания
1. Используйте виджеты из модуля tkinter и tkinter.ttk
2. Применяйте современные стили с использованием ttk
3. Обеспечьте удобство использования интерфейса
4. Обрабатывайте события взаимодействия с виджетами

## Требования к отчету
- Исходный код приложений
- Скриншоты интерфейсов
- Описание использования каждого виджета

## Критерии оценки
- Функциональность виджетов: 50%
- Удобство интерфейса: 30%
- Качество кода: 20%
