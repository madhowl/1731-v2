# Практическое занятие 17: Создание GUI с помощью Tkinter

## Цель занятия
Изучить основы создания графических интерфейсов с использованием библиотеки Tkinter, включая создание окон, виджетов, обработку событий и построение интерфейса.

## Задачи

### Задача 1: Создание базового окна (20 баллов)
Создайте приложение с основным окном:
- Установите заголовок, размеры и положение окна
- Добавьте иконку приложения
- Реализуйте обработку закрытия окна

### Задача 2: Работа с виджетами (20 баллов)
Создайте интерфейс с основными виджетами:
- Кнопки, метки, поля ввода
- Текстовые области, флажки, переключатели
- Комбинированные списки

### Задача 3: Расположение элементов (20 баллов)
Используйте разные менеджеры размещения:
- pack для простых макетов
- grid для табличных макетов
- place для точного позиционирования

### Задача 4: Обработка событий (20 баллов)
Реализуйте обработку событий:
- Клики мыши
- Нажатия клавиш
- Изменения в виджетах

### Задача 5: Комплексное приложение (20 баллов)
Создайте приложение "Калькулятор":
- Интерфейс с кнопками для цифр и операций
- Отображение результата
- Обработка математических операций

## Методические указания
1. Используйте менеджеры размещения для правильного расположения виджетов
2. Обрабатывайте события с помощью метода bind()
3. Используйте StringVar, IntVar для связи данных с виджетами
4. Следите за структурой приложения и организацией кода

## Требования к отчету
- Исходный код приложений
- Скриншоты интерфейсов
- Описание архитектуры приложения

## Критерии оценки
- Функциональность приложений: 50%
- Качество интерфейса: 30%
- Качество кода: 20%
