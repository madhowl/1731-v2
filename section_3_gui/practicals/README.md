## Цель практических заданий

Практические задания направлены на закрепление знаний по созданию графического интерфейса пользователя. Студенты научатся создавать GUI-приложения с использованием библиотек Tkinter и PyQt, работать с элементами интерфейса, обрабатывать события, использовать архитектурные паттерны MVC и MVP.
