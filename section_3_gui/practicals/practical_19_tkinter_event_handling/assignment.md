# Практическое занятие 19: Tkinter - обработка событий

## Цель занятия
Изучить механизмы обработки событий в Tkinter, включая привязку обработчиков к виджетам, работу с различными типами событий и создание интерактивных приложений.

## Задачи

### Задача 1: Обработка кликов мыши (20 баллов)
Создайте приложение с Canvas, на котором:
- Реализуйте обработку кликов левой и правой кнопки мыши
- Отображайте координаты клика
- Рисуйте фигуры по клику мыши (круги по левому клику, квадраты по правому)

### Задача 2: Обработка клавиатуры (20 баллов)
Разработайте приложение с Canvas:
- Обработка нажатий клавиш стрелок для перемещения объекта
- Обработка буквенных клавиш для изменения цвета объекта
- Обработка комбинаций клавиш (Ctrl+C, Ctrl+V и т.д.)

### Задача 3: Передача данных через события (20 баллов)
Создайте приложение с несколькими кнопками:
- Использование lambda-функций для передачи параметров в обработчики
- Передача данных о виджете через атрибуты события
- Обработка событий с передачей контекстной информации

### Задача 4: Кастомные события (20 баллов)
Реализуйте систему пользовательских событий:
- Создание и генерация собственных событий
- Обработка пользовательских событий
- Использование событий для связи между разными компонентами приложения

### Задача 5: Комплексное приложение (20 баллов)
Разработайте интерактивное приложение "Рисовалка":
- Обработка мыши для рисования
- Обработка клавиатуры для изменения инструментов
- Сохранение и загрузка рисунков
- Поддержка нескольких слоев

## Методические указания
1. Используйте метод bind() для привязки обработчиков событий
2. Работайте с объектом события для получения информации о событии
3. Используйте лямбда-функции для передачи параметров в обработчики
4. Обрабатывайте исключения в обработчиках событий

## Требования к отчету
- Исходный код программ
- Примеры работы с различными типами событий
- Объяснение передачи данных через события

## Критерии оценки
- Корректная обработка событий: 50%
- Использование различных типов событий: 30%
- Качество кода и документация: 20%
