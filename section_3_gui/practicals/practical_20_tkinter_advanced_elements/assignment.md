# Практическое занятие 20: Tkinter - продвинутые элементы

## Цель занятия
Изучить продвинутые элементы интерфейса Tkinter, включая Treeview, Progressbar, Notebook, PanedWindow, Combobox с кастомными списками и другие сложные виджеты.

## Задачи

### Задача 1: Treeview (20 баллов)
Создайте приложение с Treeview:
- Реализуйте древовидный список с несколькими уровнями
- Добавьте возможность редактирования элементов
- Реализуйте сортировку по колонкам
- Добавьте контекстное меню для операций с элементами

### Задача 2: Progressbar (20 баллов)
Реализуйте различные виды прогресс баров:
- Определенный прогресс бар с анимацией
- Неопределенный прогресс бар для длительных операций
- Множественные прогресс бары с синхронизацией
- Индикация выполнения различных задач

### Задача 3: Notebook (вкладки) (20 баллов)
Создайте интерфейс с вкладками:
- Несколько вкладок с различным содержимым
- Динамическое добавление и удаление вкладок
- Обработка событий переключения вкладок
- Сохранение состояния вкладок

### Задача 4: PanedWindow (20 баллов)
Реализуйте интерфейс с разделителями:
- Горизонтальное и вертикальное разделение
- Несколько панелей с разным содержимым
- Возможность изменения размера панелей
- Сохранение пропорций при изменении размера окна

### Задача 5: Комплексное приложение (20 баллов)
Разработайте приложение "Менеджер задач":
- Используйте все изученные элементы
- Реализуйте CRUD-операции для задач
- Добавьте фильтрацию и сортировку
- Включите визуализацию прогресса выполнения

## Методические указания
1. Используйте виджеты из модуля tkinter.ttk для современного вида
2. Обеспечьте правильное размещение элементов с использованием pack/grid/place
3. Обрабатывайте события взаимодействия с продвинутыми виджетами
4. Реализуйте адаптивный интерфейс

## Требования к отчету
- Исходный код приложений
- Скриншоты интерфейсов
- Описание особенностей каждого виджета

## Критерии оценки
- Корректная реализация виджетов: 50%
- Использование всех типов виджетов: 30%
- Качество интерфейса: 20%
