## Цель практических заданий

Практические задания направлены на закрепление знаний по архитектуре приложений и безопасности. Студенты научатся использовать виртуальные окружения, работать с системой контроля версий Git, применять архитектурные паттерны, обеспечивать безопасность приложений, работать с сетью, контейнеризацией и CI/CD.
