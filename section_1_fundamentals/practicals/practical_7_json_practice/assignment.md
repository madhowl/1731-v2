# Практическое занятие 7: Работа с JSON

## Цель занятия
Научиться работать с форматом JSON в Python, включая сериализацию и десериализацию данных, работу с внешними API, возвращающими JSON.

## Задачи

### Задача 1: Чтение JSON-файлов (25 баллов)
Создайте программу, которая:
- Читает JSON-файл с данными о пользователях
- Выводит информацию о каждом пользователе
- Обрабатывает возможные ошибки при чтении файла

### Задача 2: Запись JSON-файлов (25 баллов)
Создайте программу, которая:
- Создает структуру данных о студентах
- Записывает эту структуру в JSON-файл
- Проверяет корректность записи

### Задача 3: Работа с JSON API (25 баллов)
Используя библиотеку requests:
- Выполните запрос к любому публичному API, возвращающему JSON
- Обработайте полученные данные
- Выведите отформатированную информацию

### Задача 4: Валидация JSON-данных (25 баллов)
- Создайте схему валидации для JSON-данных
- Проверьте, соответствуют ли ваши данные схеме
- Обработайте некорректные данные

## Методические указания
1. Используйте модуль `json` для работы с JSON
2. Обязательно обрабатывайте исключения при работе с JSON
3. Следите за корректным кодированием при работе с файлами

## Требования к отчету
- Исходный код программ
- Примеры входных и выходных данных
- Описание возникших ошибок и способов их устранения

## Критерии оценки
- Корректность работы программ: 60%
- Обработка исключений: 20%
- Качество кода: 20%
