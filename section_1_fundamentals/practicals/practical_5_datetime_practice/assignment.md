# Практическое занятие 5: Работа с датой и временем

## Цель занятия
Изучить возможности модуля datetime в Python для работы с датами и временем, включая форматирование, вычисления с датами, работу с часовыми поясами и календарные вычисления.

## Задачи

### Задача 1: Основы работы с датами (20 баллов)
Создайте программу, которая:
- Получает текущую дату и время
- Выводит дату в различных форматах
- Вычисляет разницу между двумя датами

### Задача 2: Форматирование дат (20 баллов)
Реализуйте функции для:
- Преобразования строки в объект datetime
- Преобразования объекта datetime в строку
- Использования различных форматов даты и времени

### Задача 3: Работа с часовыми поясами (20 баллов)
Используйте модуль pytz или zoneinfo для:
- Работы с различными часовыми поясами
- Преобразования времени между часовыми поясами
- Учета перехода на летнее/зимнее время

### Задача 4: Календарные вычисления (20 баллов)
Создайте программу для:
- Вычисления даты через N дней/недель/месяцев
- Определения дня недели для заданной даты
- Поиска ближайшего будущего события

### Задача 5: Практическое применение (20 баллов)
Разработайте календарь встреч:
- Хранение встреч с датой и временем
- Проверка пересечений встреч
- Напоминания о предстоящих встречах

## Методические указания
1. Используйте модуль `datetime` для базовой работы с датами
2. Используйте модуль `pytz` или `zoneinfo` для работы с часовыми поясами
3. Обрабатывайте исключения при парсинге дат

## Требования к отчету
- Исходный код программ
- Примеры работы с датами и временем
- Описание использованных форматов
