# Практическое занятие 4: Использование модуля collections

## Цель занятия
Изучить возможности модуля collections в Python, включая такие структуры данных, как namedtuple, deque, Counter, OrderedDict, defaultdict, ChainMap.

## Задачи

### Задача 1: namedtuple (20 баллов)
Создайте namedtuple для представления информации о студенте:
- Создайте тип Student с полями: name, age, grade, subjects
- Создайте несколько экземпляров студентов
- Выведите информацию о студентах

### Задача 2: deque (20 баллов)
Используйте deque для реализации стека и очереди:
- Реализуйте стек с операциями push и pop
- Реализуйте очередь с операциями enqueue и dequeue
- Продемонстрируйте работу структур

### Задача 3: Counter (20 баллов)
Используйте Counter для анализа текста:
- Подсчитайте частоту слов в тексте
- Найдите самые часто встречающиеся слова
- Сравните два текста по частоте слов

### Задача 4: defaultdict (20 баллов)
Используйте defaultdict для группировки данных:
- Сгруппируйте список слов по первой букве
- Сгруппируйте оценки студентов по предметам
- Продемонстрируйте преимущества defaultdict над обычным dict

### Задача 5: OrderedDict (20 баллов)
Реализуйте LRU (Least Recently Used) кэш с использованием OrderedDict:
- Добавьте элементы в кэш
- Обновите порядок при доступе к элементу
- Удалите наименее используемый элемент при переполнении

## Методические указания
1. Импортируйте нужные классы из модуля collections
2. Используйте все возможности структур данных
3. Сравните эффективность с обычными типами данных

- Анализ преимуществ использования модуля collections
