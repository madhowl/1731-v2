# Практическое занятие 8: Работа с XML

## Цель занятия
Научиться работать с форматом XML в Python, включая парсинг, создание XML-документов, использование различных библиотек для работы с XML.

## Задачи

### Задача 1: Парсинг XML-файлов (25 баллов)
Создайте программу, которая:
- Читает XML-файл с данными о товарах
- Извлекает информацию о каждом товаре (название, цена, категория)
- Выводит отфильтрованные данные

### Задача 2: Создание XML-документов (25 баллов)
Создайте программу, которая:
- Формирует XML-документ с информацией о заказах
- Использует правильную структуру и форматирование
- Сохраняет документ в файл

### Задача 3: Работа с XML API (25 баллов)
- Выполните запрос к API, возвращающему XML-данные
- Обработайте полученные данные
- Преобразуйте XML-данные в Python-объекты

### Задача 4: Валидация XML-данных (25 баллов)
- Создайте XSD-схему для вашего XML-документа
- Проверьте, соответствуют ли данные схеме
- Обработайте некорректные данные

## Методические указания
1. Используйте модуль `xml.etree.ElementTree` для работы с XML
2. Рассмотрите альтернативные библиотеки: `lxml`, `BeautifulSoup`
3. Обязательно обрабатывайте исключения при работе с XML

## Требования к отчету
- Исходный код программ
- Примеры входных и выходных данных
- Сравнение различных библиотек для работы с XML

## Критерии оценки
- Корректность работы программ: 50%
- Использование разных методов работы с XML: 30%
- Качество кода и обработка ошибок: 20%
