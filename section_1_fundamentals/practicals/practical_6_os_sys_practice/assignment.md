# Практическое занятие 6: Работа с os и sys

## Цель занятия
Изучить возможности модулей os и sys в Python для взаимодействия с операционной системой, работы с файловой системой, переменными окружения, аргументами командной строки и системной информацией.

## Задачи

### Задача 1: Работа с файловой системой (20 баллов)
Создайте программу, которая:
- Получает текущую директорию
- Выводит список файлов и подкаталогов
- Создает новые каталоги и файлы
- Удаляет файлы и каталоги

### Задача 2: Работа с путями (20 баллов)
Реализуйте функции для:
- Объединения путей с использованием os.path.join()
- Определения абсолютных и относительных путей
- Разбора путей на компоненты

### Задача 3: Переменные окружения (20 баллов)
Используйте модуль os для:
- Получения значений переменных окружения
- Установки новых переменных
- Создания программы с конфигурацией через переменные окружения

### Задача 4: Аргументы командной строки (20 баллов)
Используйте модуль sys для:
- Получения аргументов командной строки
- Создания простой утилиты командной строки
- Обработки ошибок при запуске программы

### Задача 5: Системная информация (20 баллов)
Получите информацию о системе с помощью модуля sys:
- Версию Python
- Размеры объектов
- Пути к модулям
- Реализуйте утилиту системной диагностики

## Методические указания
1. Используйте модуль `os` для работы с файловой системой и переменными окружения
2. Используйте модуль `sys` для получения информации о системе и аргументов командной строки
3. Обрабатывайте исключения при работе с файловой системой
4. Используйте `os.path` для работы с путями к файлам

## Требования к отчету
- Исходный код программ
- Примеры работы с файловой системой
- Описание полученной системной информации

## Критерии оценки
- Корректность работы программ: 60%
- Использование различных возможностей модулей os и sys: 25%
- Качество кода и обработка ошибок: 15%
