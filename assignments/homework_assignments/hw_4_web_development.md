# Домашнее задание 4: Веб-разработка

## Цель задания
Закрепить знания по веб-разработке с использованием Flask и Django, включая создание маршрутов, шаблонов, форм, работы с API и интеграции внешних сервисов.

## Задачи

### Задача 1: Flask приложение (25 баллов)
Создайте веб-приложение "Блог" с использованием Flask:
- Главная страница с отображением списка постов
- Страница создания нового поста
- Страница просмотра отдельного поста
- Реализуйте валидацию данных формы
- Используйте шаблоны Jinja2

### Задача 2: Django приложение (25 баллов)
Создайте Django-приложение "Система обратной связи":
- Модель для хранения сообщений пользователей
- Представления для отображения и создания сообщений
- Формы для ввода данных
- Административный интерфейс для управления сообщениями

### Задача 3: Работа с API (25 баллов)
Реализуйте приложение, которое взаимодействует с внешним API:
- Используйте библиотеку requests для обращения к REST API
- Реализуйте получение, создание и обновление данных
- Добавьте обработку ошибок при работе с API
- Создайте кэширование полученных данных

### Задача 4: Создание собственного API (25 баллов)
Разработайте REST API для управления списком задач:
- Реализуйте CRUD-операции для задач
- Добавьте аутентификацию пользователей
- Используйте сериализацию данных
- Обеспечьте безопасность API

## Требования к оформлению

1. Все функции и классы должны иметь docstring
2. Код должен соответствовать PEP 8
3. Добавьте аннотации типов
4. Приложения должны быть разделены на логические модули
5. Реализуйте обработку исключений
6. Добавьте логирование запросов и ошибок
