# Домашнее задание 3: Разработка GUI-приложений

## Цель задания
Закрепить знания по созданию графических интерфейсов с использованием Tkinter и PyQt, включая создание элементов интерфейса, обработку событий, архитектуру MVC и сохранение состояния приложения.

## Задачи

### Задача 1: Tkinter приложение (25 баллов)
Создайте приложение "Заметки" с использованием Tkinter:
- Главное окно с меню
- Текстовое поле для ввода заметок
- Список сохраненных заметок
- Кнопки для добавления, редактирования и удаления заметок
- Возможность сохранения и загрузки заметок из файла

### Задача 2: PyQt приложение (25 баллов)
Создайте приложение "Калькулятор" с использованием PyQt:
- Окно с кнопками для цифр и операций
- Поле для отображения результата
- Обработка математических операций
- Использование сигналов и слотов для взаимодействия

### Задача 3: MVC архитектура (25 баллов)
Реализуйте приложение "Список задач" с использованием архитектуры MVC:
- Модель: класс для управления списком задач
- Представление: GUI-интерфейс (на ваш выбор Tkinter или PyQt)
- Контроллер: класс, связывающий модель и представление
- Реализуйте основные операции: добавление, удаление, изменение статуса задач

### Задача 4: Сохранение состояния (25 баллов)
Добавьте функциональность сохранения состояния ко всем созданным приложениям:
- Используйте JSON или pickle для сериализации данных
- Реализуйте автоматическое сохранение при закрытии приложения
- Реализуйте загрузку состояния при запуске приложения

## Требования к оформлению

1. Все классы и методы должны иметь docstring
2. Код должен соответствовать PEP 8
3. Добавьте аннотации типов
4. Приложения должны быть разделены на модули
5. Реализуйте обработку исключений
