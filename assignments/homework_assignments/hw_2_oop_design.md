# Домашнее задание 2: Объектно-ориентированное проектирование

## Цель задания
Закрепить знания по объектно-ориентированному программированию, включая наследование, полиморфизм, инкапсуляцию, абстрактные классы, магические методы и паттерны проектирования.

## Задачи

### Задача 1: ООП концепции (25 баллов)
Создайте иерархию классов для системы управления библиотекой:
- Абстрактный класс `Item` с базовыми свойствами (название, автор, год выпуска)
- Классы-наследники `Book`, `Magazine`, `DVD` с уникальными свойствами
- Класс `LibraryMember` для представления читателя
- Класс `Library` для управления книгами и читателями

Реализуйте магические методы `__str__`, `__repr__`, `__eq__` для всех классов.

### Задача 2: Паттерн "Одиночка" (25 баллов)
Реализуйте паттерн Singleton для класса `DatabaseConnection`, который управляет подключением к базе данных. Убедитесь, что:
- Только один экземпляр класса может существовать
- Класс потокобезопасен
- Реализуйте методы подключения и отключения от базы данных

### Задача 3: Паттерн "Фабрика" (25 баллов)
Создайте фабрику `DocumentFactory`, которая создает различные типы документов:
- `PDFDocument`, `WordDocument`, `ExcelDocument`
- Фабрика должна принимать тип документа и возвращать соответствующий объект
- Все документы должны реализовывать общий интерфейс с методами `open()`, `save()`, `close()`

### Задача 4: Паттерн "Наблюдатель" (25 баллов)
Реализуйте паттерн Observer для системы уведомлений:
- Класс `NewsAgency` как издатель новостей
- Классы `SMSSubscriber`, `EmailSubscriber`, `PushSubscriber` как подписчики
- При публикации новости все подписчики должны быть уведомлены

## Требования к оформлению

1. Все классы и методы должны иметь docstring
2. Код должен соответствовать PEP 8
3. Добавьте аннотации типов
4. Напишите простые тесты для демонстрации работы паттернов
