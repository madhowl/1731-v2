# Список библиотек

## Основные библиотеки Python

### Стандартная библиотека
- `os` - работа с операционной системой
- `sys` - системные параметры и функции
- `datetime` - работа с датой и временем
- `collections` - расширенные коллекции
- `json` - работа с JSON
- `xml.etree.ElementTree` - работа с XML
- `itertools` - итераторы
- `functools` - функциональные инструменты
- `argparse` - разбор аргументов командной строки
- `logging` - логирование

### Внешние библиотеки
- `requests` - работа с HTTP-запросами
- `flask` - микрофреймворк для веб-разработки
- `django` - веб-фреймворк
- `sqlalchemy` - ORM для работы с базами данных
- `sqlite3` - работа с SQLite
- `tkinter` - создание GUI-приложений
- `pandas` - работа с данными
- `numpy` - численные вычисления
- `matplotlib` - построение графиков
- `pillow` - работа с изображениями

## GUI-библиотеки
- `tkinter` - стандартная библиотека для GUI
- `PyQt5/PyQt6` - мощный фреймворк для GUI
- `PySide2/PySide6` - альтернатива PyQt

## Работа с базами данных
- `sqlalchemy` - ORM
- `psycopg2` - PostgreSQL
- `pymysql` - MySQL
- `pymongo` - MongoDB

## Тестирование
- `unittest` - встроенный модуль тестирования
- `pytest` - сторонний фреймворк для тестирования
- `mock` - mocking-библиотека

## Дополнительные инструменты
- `virtualenv` - виртуальные окружения
- `pipenv` - управление зависимостями
- `black` - форматирование кода
- `flake8` - проверка качества кода
- `mypy` - статическая проверка типов
